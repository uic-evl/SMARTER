1. use anaconda for better and easy creation of environment
2. open anaconda navigator -> environment -> create -> check python and R
3. Now open anaconda prompt
4. change to your newly created env
conda activate youEnvName
4. install the packages you need 
     a. rpy2
conda install -c r rpy2
     b. flask
conda install -c anaconda flask
     c. flask-cors
conda install -c anaconda flask-cors
     d. pandas
conda install -c anaconda pandas
 	e. numpy
conda install -c anaconda numpy

	g. add survminer
